Practice Sets Help

- Play the practice set in the browser.

- Schedule the practice set for later via mobile call.

What are Practice Sets

Practice Sets are collections of questions that can be played to simulate a live language tandem. Playing starts a question / answer session where each question in the set is played with randomly selected audio from the Tandem network. After each question you will be prompted to replay to the question and your answer will be recorded and stored for grading by the Tandem community.

Scheduling a Practice Set

Scheduling a Practice Set allows you to practice via mobile using a standard mobile phone call. Scheduled sessions play exactly as they do when played in the browser, however, there will be a short beep after each question indicating when your answer will start recording. Just as in browser play, your answers will be recorded and stored for grading.

Standard mobile to mobile rates will apply during Scheduled practice session**
